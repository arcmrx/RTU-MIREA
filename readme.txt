## Команды в терминале:

- `pwd` - Вывести на экран текущую рабочую директорию
- `cd` - Сменить текущую директорию
- `~` - Обозначение домашней директории
- `ls` **-** Вывести содержимое директории
- `..` - Чтобы вернуться в **родительскую директорию** — то есть на уровень выше
- `ls -a` - Напечатать содержимое директории со скрытыми файлами
- `clear` - Очистка терминала
- `touch` - Создание файла (с указанием расширения)
- `mkdir` - Создания директорий
- `mkdir -p` - Создать целую структуру директорий одной командой
- `cp` - Копирование файлов(`cp что_копируем куда_копируем`)
- `mv` **-** Перемещение файлов и папок(`mv перемещаем перемещаем`)
- `cat` - Чтение файлов(.txt)
- `rm` - Удаление файлов
- `rmdir` - Удаление пустой директории
- `rm -r` - Удаление директории с файлами
- `&&` - Выполняйте сразу несколько команд
- `↑` `↓` - Вызов команд из буфера
- `Tab` - Автозаполнение
